## Setup
To run this project, install it locally using npm:

```
$ git clone https://github.com/atlp-rwanda/trojans-ec-bn.git
$ npm install
$ npm start
```


### To run migrations:

```
$ npm run migrate
$ npm run down (for undo migration)
```

### To run seed for add dump data:

```
$ npm run seed
$ npm run undo-seed (for undo seed)
```
